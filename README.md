Walkthrough: Application de Révisions IFSI 🚀
Ce document résume les nouveautés implémentées dans l'application.

1. Page d'Accueil à double niveau
L'application s'ouvre sur une Landing Page où l'on choisit d'abord son Unité d'Enseignement (UE 2.8 ou UE 2.7).
Ce n'est qu'après ce choix qu'on accède à la page des Thèmes spécifiques à l'UE sélectionnée.
2. Nouveaux Thèmes et UE 2.7 (Mise à jour majeure !)
La base de données a été migrée vers data.js pour accueillir une base de questions massive.

UE 2.8 (Obstructif/Cardio) : Respiratoire, Digestif, Urinaire, Cardio-Vasculaire, Lecture ECG.
UE 2.7 (Dégénératif/Organique) : Refonte complète avec 12 sous-thèmes spécifiques (Parkinson, Alzheimer, SEP, Vieillissement, IRC, Diabète & Gestationnel, Complications du Diabète, Arthrose, Troubles Visuels, Troubles Auditifs, Insuffisance Respiratoire (2.7), Homéostasie et pH).
Chaque UE possède maintenant son propre bouton Mix Complet qui regroupe toutes ses spécialités sans croiser l'autre UE.
3. Pédagogie : Encart d'explications
En cas de mauvaise réponse, le quiz se met en attente.
Un Encart d'explications stylisé s'affiche pour donner la physiopathologie à retenir, avec validation manuelle pour continuer.
4. Ergonomie : Bouton Accueil & Pause
Bouton Accueil : Un bouton 🏠 permet de revenir au menu central à tout moment.
Sauvegarde et Reprise : Quitter le quiz en cours le met en pause sans perdre la progression. Un gros bouton vert permettra de reprendre directement depuis la page d'accueil.
